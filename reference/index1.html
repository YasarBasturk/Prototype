Processing Information

When you upload an image:

  1. The image is preprocessed to enhance quality
  2. Text is automatically detected using OCR
  3. Cell detection identifies table cells in the document
  4. Text spanning analysis combines OCR and cell detection
  5. Template matching identifies editable vs non-editable text
  6. You can edit the handwritten text elements
CELL DETECTION: Identifies tables and their cells in documents
TEXT SPANNING: Associates text with table cells and handles text that spans multiple cells
